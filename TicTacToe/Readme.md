# Tic-Tac-Toe Game-

A simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript. This project demonstrates core web development concepts such as DOM manipulation, event handling, and win-checking logic.


# Features-

Turn-based gameplay (X & O)

Winner detection using predefined win patterns

Reset & New Game functionality

Clean and simple UI


# Technologies Used -

HTML – Game structure

CSS – Styling for layout & UI

JavaScript – Game logic, event listeners, and DOM interactions


# Installation -

Clone the repository:

git clone https://github.com/NainaKalra/JS-Mini-Projects.git

Navigate into your TicTacToe folder and open index.html in your browser.


# How to Play -

Click any empty box to place your mark.

Players alternate between O and X.

When a winning combination is detected, the game announces the winner.

Press Reset or New Game to play again.
